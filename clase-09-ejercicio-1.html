Clase 09: Arrays III - ECMA6 -Ejercicio 1

Recorriendo arrays

Creemos un array de datos (países, bandas, frutas, etc).

Tenemos el botón "Mostrar datos", que al hacerle click debe ejecutar la función RecorrerArray que reciba como argumento el array que creamos.

Cada ítem del array debe irse mostrando en un li, que se guardará en un ul que se deberá ir agregando a la variable info.

Dicha variable debe ser pasada como argumento a la función "MostrarInfo" (una función ya creada que se encarga de mostrar el código generado … que por supuesto les voy a explicar para que entiendan que hace).

Tomando como base la función anterior, creamos la función RecorrerArray2.

Esta función, ahora, además debe recibir un segundo argumento, que será desde dónde comenzará a recorrer el array.

El ingreso de este segundo parámetro será mediante prompt(). Por defecto 0.

Validar y contemplar que pasa si:

  • No es un número.
  • Si el número es menor a 0.
  • Si el número supera la longitud del array.

Tomando como base la función anterior, creamos la función RecorrerArray3.

Esta función, ahora, además debe recibir un tercer argumento, que será hasta dónde recorrerá el array.

El ingreso de este segundo parámetro será mediante prompt(). Por defecto la longitud del array.

Validar y contemplar que pasa si:

  • No es un número.
  • Si el número es menor a 0.
  • Si el número supera la longitud del array.
  • Si este valor es menor al inicial.
